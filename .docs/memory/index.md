## Usage Guide

### For All Agent Modes

1. Reference memory patterns when encountering similar challenges
2. Apply proven approaches while respecting current appetite constraints
3. Contribute new learnings when completing successful implementations
4. Update complexity scoring based on actual resource usage

### For Appetite Planning

- Memory patterns inform realistic resource allocation
- Successful approaches reduce risk in appetite estimation
- Failure patterns help define appropriate circuit breakers
- Complexity scoring guides 70/30 decision boundaries

### Memory Structure

Each memory file includes:

- **Pattern Name**: Clear identifier for the approach or learning
- **Context**: Appetite constraints and project conditions when pattern was applied
- **Implementation**: Specific steps and techniques used
- **Outcomes**: Results achieved and resource consumption
- **Lessons**: Key insights for future application
