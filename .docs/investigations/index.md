## Usage Guide

### For Debug Mode Agents

1. Search this index for similar problem patterns before starting investigation
2. Follow established investigation templates and appetite constraints
3. Document findings with clear root cause analysis and resolution steps
4. Update this index when completing investigations

### For Prevention

- Investigation findings inform future circuit breaker definitions
- Patterns help identify early warning signs of similar issues
- Resolution strategies guide appetite allocation for debugging work

### Investigation Structure

Each investigation includes:

- **Problem Summary**: Clear description of symptoms and impact
- **Root Cause Analysis**: Systematic investigation findings
- **Resolution Applied**: Specific steps taken to fix the issue
- **Prevention Measures**: How to avoid similar issues in future
- **Appetite Impact**: Time spent vs. planned allocation
