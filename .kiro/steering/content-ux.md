# Content & UX Guidelines

## Brand Voice Requirements
- **Tone**: Conversational, supportive, inclusive - use "Hi mama", "lovely", "we get it"
- **Community**: Always use "M.O.M.unity" and "stronger together" messaging
- **Forbidden**: Never use "bounce back", "pre-baby body", "six-pack pressure"
- **Replacements**: Use "come as you are", "start where you are", "your journey"

## Required Content Patterns

### Exact Headlines to Use
```
"Welcome to Mood Over Muscle 💗"
"Hi mama, I'm Emily"
"Start Your FREE Session"
"Book Your FREE Session"
"Join M.O.M.unity"
```

### Testimonial Format (Strict)
```
"[Quote]" - [First Name] [Last Initial]
Example: "No pressure, just support!" - Sarah M.
```

### Benefit Messaging Pattern
```
[Physical benefit] + [Emotional outcome]
Example: "We rebuild from the inside out – pelvic floor, core, confidence, and calm"
```

## UX Architecture

### Required Page Structure
1. Hero: Emotional hook + FREE emphasis + CTA
2. About: Personal story + credentials + trust signals
3. How It Works: 4-step process
4. Gallery: Community photos + social proof
5. Contact: Multiple contact methods + location

### Form Design Rules
- Multi-step with progress indicators
- Real-time validation with helpful errors
- Constantly emphasize FREE (no payment/commitment)
- Ask personal questions (goals, preferences)
- Mobile-first: min 44px touch targets

### FREE Session Emphasis (Critical)
- Use green gradient: `from-green-500 to-emerald-500`
- Visual elements: badges, sparkle icons, pulse animations
- Risk reversal: "No payment", "No commitment", "No credit card"
- Repeat across hero, navigation, forms, CTAs

## Accessibility Standards
- WCAG AA color contrast compliance
- Semantic HTML with proper heading hierarchy (h1, h2, h3)
- ARIA labels for interactive elements
- Keyboard navigation support
- Descriptive alt text for all images
- Screen reader compatible

## Mobile-First Requirements
- Touch-friendly buttons (min 44px)
- Thumb-reachable navigation
- Collapsible menus
- Responsive images with proper aspect ratios
- Large input fields with clear labels

## Trust Building Elements
- Display Emily's certifications prominently
- Include personal/family photos
- Show physical address and contact info
- Use real client testimonials with photos
- Emphasize "100% risk-free" messaging

## Content Maintenance Rules
- Update seasonal messaging (New Year, holidays, summer)
- Rotate gallery photos regularly
- Refresh testimonials every few months
- Keep service offerings current
